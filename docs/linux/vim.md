#### Vim 常用命令：

##### 1. 编辑模式

   - i 在当前光标处进入插入模式

   - a 在当前光标后进入插入模式

   - o 在当前行下方插入新行并进入插入模式

   - I 在当前行的行首进入插入模式

   - A 在当前行的行尾进入插入模式

   - O 在当前行上方插入新行并进入插入模式

   - r 替换当前光标所在字符

   - R 进入替换模式，直到按下 ESC 键


##### 2. 移动光标

   - h 左移一个字符

   - j 下移一行

   - k 上移一行

   - l 右移一个字符

   - w 移动到下一个单词开头

   - e 移动到下一个单词结尾

   - b 移动到上一个单词开头

   - 0 移动到当前行行首

   - $ 移动到当前行行尾

   - gg 移动到文件第一行

   - G 移动到文件最后一行

   - :n 跳转到第 n 行


##### 3. 操作文本

   - yy 复制当前行

   - dd 删除当前行

   - p 粘贴

   - u 撤销上一次操作

   - Ctrl+r 重做上一次被撤销的操作

   - :s/old/new/g 替换 old 为 new，g 表示全局替换，如果不加 g 只会替换每行第一个匹配的字符串


##### 4. 其他命令

   - :q 退出 Vim

   - :w 保存修改

   - :wq 保存并退出

   - :q! 强制退出不保存修改
