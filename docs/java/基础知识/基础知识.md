##### 1. Class和ClassLoader中的getResource（）和getResourceAsStream之间的区别

     ```
1. ClassLoader.getResource(String name)
只能够从classpath根目录开始匹配获取资源,写法:
classloader.getResource(“test.properties”)
2. Class.getResource(String name)
从当前Class所在包的路径开始匹配资源,写法:
Class.getResource("")
从classpath根路径开始获取资源，写法:
Class.getResource("/")
     ```
