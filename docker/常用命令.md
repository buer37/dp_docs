#### 常用命令
##### 重启service
```bash
docker service update --image registry.cn-beijing.aliyuncs.com/private_deploy/wechat-center:private-20241210-250811 yunke_wechatweb

docker service update --image registry.cn-beijing.aliyuncs.com/private_deploy/mq-kafka:private-20241210-250811 yunke_mq-kafka 
```

##### 查看service日志
--tail 100 表示查看最后100行日志
```bash
docker service logs --tail 100 yunke_mq-kafka
```

--f 表示持续查看日志
```bash
docker service logs --f yunke_mq-kafka
```

--since 2025-08-11 表示查看2025-08-11之后的日志
```bash
# 北京时间（UTC+8）可加 +08:00，或用 UTC 时间（加 Z 后缀）
docker service logs --since "2025-08-11T17:00:00+08:00" yunke_mq-kafka
```

相对时间（更简单，推荐）
直接指定 “多久之前” 的相对时间，无需关心格式：

- 查看 1小时前至今的日志
```bash
docker service logs --since "1h" yunke_mq-kafka
```

- 查看 30分钟前至今的日志   
```bash
docker service logs --since "30m" yunke_mq-kafka
```

- 查看 2小时30分钟前至今的日志
```bash
docker service logs --since "2h30m" yunke_mq-kafka
```

##### 查看service状态
```bash
docker service ps yunke_wechatweb
```
