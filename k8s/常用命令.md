### k8s常用命令


#### deployment
##### 查看
```bash
kubectl get deployment -n yunke-prod
kubectl get deployment -n yunke-prod -o wide
```

##### 编辑
```bash
kubectl edit deployment -n yunke-prod {deployment-name}
kubectl edit deployment -n yunke-prod mq-kafka
kubectl edit deployment -n yunke-test mq-kafka
kubectl edit deployment -n yunke-prod wechat-center
```


##### 滚动发版
```bash
kubectl rollout restart deployment -n yunke-prod {deployment-name}
kubectl rollout restart deployment -n yunke-prod mq-kafka
kubectl rollout restart deployment -n yunke-prod wechat-center
```

#### 查看滚动发版状态
```bash
kubectl rollout status deployment -n yunke-prod {deployment-name}
kubectl rollout status deployment -n yunke-prod mq-kafka
kubectl rollout status deployment -n yunke-test mq-kafka

kubectl rollout status deployment -n yunke-prod wechat-center

```

#### 查看pod
```bash

kubectl get pods -n yunke-prod
kubectl get pods -n yunke-prod | grep mq-kafka
kubectl get pods -n yunke-test | grep mq-kafka
kubectl get pods -n yunke-prod | grep wechat-center
```

#### 查看pod详情
```bash
kubectl describe pod -n yunke-prod {pod-name}
kubectl describe pod -n yunke-prod wechat-center-7c86b85f84-m2qm8
```


#### 查看pod日志
```bash 
kubectl logs -n yunke-prod -f {pod-name}
kubectl logs -n yunke-prod -f wechat-center-7c86b85f84-m2qm8
kubectl logs -n yunke-test -f wechat-center-7c86b85f84-m2qm8
```
